# Benchmark Analysis

## Objective

Compare mutual fund performance against benchmark indices.

## Benchmarks Used

- Nifty 50
- Nifty 100
- BSE SmallCap

## Methodology

1. Extract benchmark index data.
2. Calculate benchmark returns.
3. Compare benchmark returns with fund returns.
4. Measure Alpha and Beta.
5. Evaluate outperformance.

## Findings

- Mid-cap funds generated higher Alpha than large-cap funds.
- Several funds outperformed Nifty 50 over the 3-year period.
- Higher Sharpe Ratio funds generally delivered superior risk-adjusted returns.

## Conclusion

Benchmark comparison helps investors evaluate whether fund managers create excess returns beyond market performance.
