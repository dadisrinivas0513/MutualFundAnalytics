# Mutual Fund Analytics Platform

## Bluestock Fintech Internship Capstone Project

### Project Overview

This project focuses on building a Mutual Fund Analytics Platform using publicly available Indian mutual fund datasets. The platform performs data ingestion, ETL processing, exploratory analysis, risk metric calculation, benchmark comparison, and dashboard visualization.

---

## Objectives

- Build an automated ETL pipeline
- Store data in a relational database
- Perform exploratory data analysis
- Calculate risk and performance metrics
- Compare fund performance against benchmarks
- Develop an analytics dashboard

---

## Dataset Sources

- AMFI India
- mfapi.in
- Public benchmark index datasets

---

## Technologies Used

- Python
- Pandas
- NumPy
- SQLite
- SQLAlchemy
- Matplotlib
- Seaborn
- Streamlit
- Git & GitHub

---

## ETL Pipeline

1. Extract raw CSV datasets
2. Clean and validate records
3. Transform columns and formats
4. Load into SQLite database
5. Generate analytical outputs

---

## Exploratory Data Analysis

Key analyses performed:

- Fund House AUM Analysis
- Monthly SIP Trend Analysis
- Category Inflow Analysis
- Scheme Performance Analysis

---

## Risk Metrics

Calculated metrics include:

- Alpha
- Beta
- Sharpe Ratio
- Sortino Ratio
- Maximum Drawdown

---

## Benchmark Analysis

Benchmarks used:

- Nifty 50
- Nifty 100
- BSE SmallCap

Performance comparison was conducted to identify fund outperformance and risk-adjusted returns.

---

## Dashboard

The Streamlit dashboard provides:

- Dataset Overview
- AUM Analysis
- SIP Trends
- Category Inflows
- Sharpe Ratio Rankings

---

## Conclusion

The Mutual Fund Analytics Platform successfully demonstrates a complete financial analytics workflow from raw data ingestion to business intelligence reporting. The project highlights practical applications of data engineering, analytics, and dashboard development in the Fintech domain.
